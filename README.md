# ✦ Alice — Portfólio Interativo

Uma experiência visual inspirada em *Alice no País das Maravilhas*, construída como portfólio criativo com HTML, CSS e JavaScript.

---

## 🕳️ A experiência

Você cai. A tela escurece. Algo espera no escuro.

Alice é um projeto de narrativa interativa onde cada clique revela um novo momento da história, animações, efeitos de luz, sons e transições contadas em código.
